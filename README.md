# taomi-tools  
淘米辅助工具  
为方便域名爱好者打造的辅助型工具。支持万网、聚名网、易名中国、爱名网（可能会不定期更新）；  
功能如下：    
1、域名列表、域名出售详情页增加以下按钮：  
- 天眼查  
- 估价  
- 查域名注册情况  
- 站长工具（爱站）综合查询     
3、搜索时自动补上".com"后缀（仅支持聚名网）；    
4、可以使用“一键检测注册状态”按钮（仅支持聚名网）；
